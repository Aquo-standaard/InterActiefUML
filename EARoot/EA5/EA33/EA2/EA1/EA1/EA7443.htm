The initial (root) class of the LCML is the LC_LandCoverClassificationSystemMetaDescription metaobject. This metaobject is composed of all the land cover elements that when instantiated produce the classes that make up a classification system as given by LC_LandCoverClassificationSystem. <br/><br/>This class has one relationship metaS<i>ystemCollection</i>. LC_ LandCoverClassificationSystemMetaDescription metaobject is an aggregation of the land cover metaobjects LC_LandCover.<br/><br/>
