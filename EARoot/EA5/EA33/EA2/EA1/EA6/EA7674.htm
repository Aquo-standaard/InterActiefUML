The LC_PermittedPosRealRange metaclass is a subtype of LC_PermittedRealValueType. It is a metaclass for a positive real number value range permitted in the metalanguage where the attribute minValue and maxValue may be instantiated as a pair of real number types using the Real type from the basic Primitive Types package in ISO 19103, with the constraints:<br/><br/>   {maxValue #gt; minValue  <br/>and<br/>   minValue #gt;= 0.0<br/>and <br/>   maxValue #gt;= 0.0}.<br/><br/>
