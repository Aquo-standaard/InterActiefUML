The three subtypes to RE_RegisteredItem are LC_ClassCharacteristicDescriptorItem, LC_ElementCharacteristicDescriptorItem and LC_ElementDescriptorItem. These subtypes take the same form and include the following attributes:<br/><br/><ul>
<li>A definition of the registered LCML metaclass;</li><li>A description of any metaclass attributes to the LCML metaclass;</li><li>A reference to the parent class in the LCML metamodel (either from the standard or another registered class) together with an indication of the type of relationship such as: subtype, component, enumeration;</li><li>A reference to any LCML metaclass which this metaclass supersedes (i.e. causes to be deprecated). Note that LC_Element classes may only be extended; that is, an instance of the registered item may be superseded only by another instance that contains all of the information from the superseded registered item.</li></ul>
