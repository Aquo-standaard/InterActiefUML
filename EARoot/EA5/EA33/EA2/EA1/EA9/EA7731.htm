This example is a translation of a class from the CORINNE LCCS. The CORINNE description is "most of the land is covered by structures and the transport network. Building, roads and artificially surfaced areas cover more than 80% of the total surface". <br/>The model  shows that two horizontal patterns have been applied one with the LCML element Non Linear Surface and the other with the LCML element Linear Surface. The use of two Horizontal Patterns implies that this combination of non linear and linear elements for a “unicum” that will be expressed in the same way at any scale used<br/><br/>
